Projects

Web app - BooksExchange
Feb 2018 - Current

A platform inspired by the BookCrossing initiative and the trends of shared economy. The main idea is to give people the possibility to connect with others that have a given book in a given region and exchange one book for another without actually putting any monetary value in the exchanged books. This way small communities can be created within cities/regions/counties or even within a company/school or university. The biggest goal is to create the biggest open library that enables the books to travel all over the world. Future thoughts for the platform is to implement a blockchain (possibly Hyperledger) solution so that all exchanges can be kept in a ledger.
For the front-end we are using React.js with the material design MDL Lite library for styling the interface. For the back-end we are using ASP.NET Core with EF Core and SQL Server.
